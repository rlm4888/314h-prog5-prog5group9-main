# Boggle Game

## Overview

This Java program implements the Boggle game with recursive algorithms and various dictionary strategies. It allows players to find words on a 4x4 grid following specific rules.

## Features

- Randomly generates Boggle boards
- Validates and scores words based on game rules
- Testing report and potential code revisions

## Usage

1. Compile the code: `javac assignment/Boggle.java`
2. Run the game: `java assignment.Boggle`

## Game Rules

- Identify words on the grid
- Words must be at least four letters long
- Path must be connected horizontally, vertically, or diagonally
- Each cube can only be used once per word
